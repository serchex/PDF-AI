Este proyecto aun no esta terminado ya que los modelos utilizados GANs no son suficientes para mejorar la calidad de las imagenes.
Independientemente de la calidad de las imagenes, el modelo es capaz de tomar contexto sobre las imagenes o texto filtrado mediante REGEX,
sabe identificar roles dentro del texto y en caso de haber documentos diferentes pero que pertenecen a una misma persona, el modelo es capaz
de reconocer en que documentos esta dicha persona y que relacion hay entre si.
